Conditionnel & Subordonnées

Français · 4e · Cours + Exercices

📖 Définition
📘 Leçon

Le conditionnel est un mode qui évoque des actions incertaines ou non accomplies.

Il possède deux temps : le présent et le passé.

⚙️ Formation
Présent = radical du futur + terminaisons de l'imparfait (-ais, -ais, -ait, -ions, -iez, -aient)
Passé = auxiliaire au présent du conditionnel + participe passé
```
Temps AimerFinirÊtrePouvoir
Futur de l'indicatif j'aimeraitu finirasil seranous pourrons
Présent du conditionnel j'aimeraistu finiraisil seraitnous pourrions
Passé du conditionnel j'aurais aimétu aurais finiil aurait éténous aurions pu
```
💡 Valeurs (à quoi sert le conditionnel ?)

🟠 Présent du conditionnel — 4 valeurs

Futur dans le passé Il exprime une action à venir dans un contexte passé.
Je croyais qu'elle se qualifierait.
Fait incertain Il exprime un doute par rapport à l'information donnée.
Le policier serait dans un état critique.
S'il venait, je serais ravie.
Fait imaginaire / rêvé On habiterait dans un palais, on aurait des tapis volants…
Demande polie / souhait Je voudrais deux croissants, s'il vous plaît.
J'aimerais que vous parliez moins fort.

🟣 Passé du conditionnel — 3 valeurs

Fait incertain dans le passé Ils seraient rentrés chez eux, mais auraient eu un accident.
Demande (très) polie J'aurais voulu vous parler.
Regret J'aurais voulu être artiste.
🧠 Astuce : futur ou conditionnel à la 1ère personne ?
Change "je" par "il" : si ça finit en -ait → conditionnel · si ça finit en -a → futur
"je chanterais" → "il chanterait" → conditionnel ✓
🔗 C'est quoi une complétive ?
📘 Leçon — à retenir

Une complétive est une subordonnée dont la fonction est généralement COD du verbe principal.

Elle fait partie du groupe verbal et ne peut pas être supprimée sans changer le sens de la phrase.

Il existe 2 types de complétives :

🟢 Conjonctive
Introduite par que
Je pense qu'il viendra.
🟣 Interrogative indirecte
Introduite par si, comment, où…
Je demande s'il viendra.
💡 En résumé : complétive = terme général. Conjonctive et interrogative indirecte sont les deux sortes de complétives. Dans les exercices, le manuel demande soit "complétive" (terme général), soit le type précis.
🟢 Type 1 — La conjonctive

🟢 Subordonnée conjonctive (= complétive introduite par "que")

Mot introducteur : la conjonction de subordination que

Elle vient juste après un verbe de parole, de pensée, de sentiment ou de perception :

Je lui avoue que je l'aime.  → verbe de parole
Je pense que tout ira bien.  → verbe de pensée
J'aime qu'on passe du temps ensemble.  → verbe de sentiment
J'ai vu qu'il était gêné.  → verbe de perception
🔑 Test pour la reconnaître : remplace la subordonnée par le mot "cela".
"Je pense cela." → ça fonctionne → c'est une conjonctive ✓

⚠️ Attention au mode :
→ fait réel → indicatif : Je sais qu'elle est malade.
→ souhait → subjonctif : Je souhaite qu'elle guérisse.
🟣 Type 2 — L'interrogative indirecte

🟣 Subordonnée interrogative indirecte (= complétive qui pose une question)

Elle vient après un verbe d'interrogation ou d'ignorance : se demander, ignorer, ne pas savoir…

2 cas :

Question totale (oui/non) → introduite par si
Je me demande [si elle viendra]. → = "Viendra-t-elle ?"
Question partielle → introduite par où · quand · comment · combien · pourquoi · qui · quel…
Je me demande [qui sera là]. → = "Qui sera là ?"
J'ignore [combien nous serons]. → = "Combien serons-nous ?"
🔑 Test pour la reconnaître : transforme-la en question directe avec "?".
"se demander si elle viendra" → "Viendra-t-elle ?" → complétive interrogative indirecte ✓
⚠️ Le grand piège : relative vs complétive

La vraie question à se poser :

🔵 Relative
Il y a un nom juste avant
la subordonnée

La leçon que nous faisons
↑ "leçon" = nom antécédent
🟢 Complétive
Il y a un verbe juste avant
la subordonnée

Je pense que cette leçon…
↑ "pense" = verbe
⚠️ Le piège du "que" : il peut être pronom relatif (relative) OU conjonction (complétive) !
Regarde ce qui est juste avant :
→ un nom → relative   |   → un verbe → complétive

Les 4 exemples du manuel :

La leçon [que nous faisons] est simple. → Relative ("leçon" est juste avant → nom antécédent)
Je pense [que cette leçon est simple]. → Complétive conjonctive ("pense" est juste avant → verbe)
La personne [qui a rédigé cette leçon] se moque de nous. → Relative ("personne" est juste avant → nom antécédent)
Je me demande [qui a rédigé cette leçon]. → Complétive interrogative indirecte ("demande" est juste avant → verbe)
📊 Tableau récapitulatif
Type Mot introducteur Ce qui est juste avant Test rapide
Relative qui, que, dont, où, lequel… Un nom (antécédent) Quel nom précède ?
Complétive conjonctive que (conjonction) Un verbe Remplace par "cela"
Complétive interrogative ind. si, comment, quand, pourquoi, où, quel… Un verbe de demande Transforme en question "?"
